V této složce jsou všechny instalace, které jsou potřebné ke spuštějí programu. Důvod uložení instalačních souborů je jejich možné odstranění z originálních webových stánek. Primárně instalace naoqi je tímto ohrožena a již při tvorbě tohoto repozitáře bylo její nalezení náročné.

Při možné aktualizaci těchto instalací je nutné udržet originální verzy 32bit.

Odkazy pro stažení dalších nutností:

https://get.videolan.org/vlc/3.0.20/win32/vlc-3.0.20-win32.exe

Postup instalací:
1) Instalace Python 2.7
  Pokud je již nainstalován Python 3.X, je nutné instalovat do verze 2.7. Jednou z možností je úprava variable Path. Viz. https://www.imatest.com/docs/editing-system-environment-variables/#Windows
2) Instalace VLC a naoQi
3) Instalace knihoven pomocí PIP
